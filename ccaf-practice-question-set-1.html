Practice Question Set 1

40 questions weighted toward Domains 1 and 3 — the two heaviest on the blueprint, and the two richest in exact-flag recall. Each is tagged with its domain so you can score yourself the way the real report does.

These drill what the modules teach in prose but rarely ask you to retrieve: the exact flag, the exact frontmatter key, the exact command name. A wrong answer is a pointer back to the module section it came from.

Mix: D1 ×12 · D2 ×1 · D3 ×24 · D4 ×2 · D5 ×1. Wrong answers explain why and reopen for a retry — same behaviour as the module checks. Nothing here is tracked; this page saves no progress.

Q4You have finished an expensive analysis of a module and now want to develop two competing refactoring strategies from that same baseline, without either exploration contaminating the other. Which capability fits?D1

Q5You want to resume last week’s refactoring session, but the team has since merged substantial changes across the files it analysed — most of its tool results now describe code that no longer exists. What is the reliable approach?D1

Q6Your coordinator invokes the synthesis subagent with the prompt “Synthesise the research findings.” The search subagent gathered excellent material, but synthesis output is generic and references none of it. What is wrong?D1

Q7Your coordinator invokes the web search subagent, waits for it to finish, then invokes document analysis — doubling latency even though the two are independent. How do you run them concurrently?D1

Q8A customer’s single message raises three separate concerns: a damaged-item refund, an address change, and a missing loyalty credit. How should the agent handle it?D1
